void wxPdfLayerMembership::SetVisibilityPolicy ( wxPdfOcgPolicy policy)

Set visibility policy.

Sets the visibility policy for content belonging to this membership dictionary. Possible values are wxPDF_OCGPOLICY_ALLON, wxPDF_OCGPOLICY_ANYON, wxPDF_OCGPOLICY_ANYOFF and wxPDF_OCGPOLICY_ALLOFF. The default value is wxPDF_OCGPOLICY_ANYON.
